bool menu_activate_item ( uint8_t  id)

Activate an item of the current menu. If the item is another menu; it is made active and displayed. If the item is an action; the action is executed.

Parameters
idThe number of the element selected in the current menu.
Returns
false if the item does not exist. true if the display/action was succesful.
