const uint8_t *  font = FONT_MEDIUM

Display a single line message to the Box' oled display. Only works on a limited set of fonts. These fonts have been analysed for the proper positioning; other fonts will not work.

Parameters
messageA c string containing the message (null terminated).
fontThe font to be used. Should be one of the fonts listed in this header file. Defaults to Medium.
